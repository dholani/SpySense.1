# Network Security Monitoring Tool

## Project Overview
An offline-capable network security monitoring tool that tracks connections, provides physical addresses for IP locations, and maintains comprehensive logs. The tool helps users monitor their network for unauthorized access and suspicious activities.

## User Preferences
- Light color theme preferred
- Comprehensive logging capabilities required
- Offline/online functionality needed
- VPN detection and traceroute analysis
- Address resolution for IP locations

## Project Architecture
- **Frontend**: React with TypeScript, Vite, shadcn/ui components
- **Backend**: Express.js with TypeScript
- **Storage**: In-memory storage with persistent logging
- **Styling**: Tailwind CSS with light theme focus
- **Data Model**: IP tracking, connection logs, geolocation data

## Core Features
- Real-time network connection monitoring
- IP address logging and geolocation
- VPN detection capabilities
- Traceroute analysis
- Historical data retention
- Offline/online synchronization
- Physical address resolution
- Comprehensive security logs
